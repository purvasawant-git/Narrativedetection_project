Algorithmic Causality in Financial Narratives.

A personal data science exploration of 6,267 news articles and stock volatility, using Unsupervised Machine Learning and Vector Autoregression.

The Big Pivot

My original hypothesis was simple: Does breaking news drive stock market panic?

The Expected Pattern

I assumed negative media narratives would cause immediate drops in the target stock price. I believed the news cycle was the driver of market behavior.

What the Data Actually Proved

After running rigorous **Granger Causality** tests on 156 weeks of aligned data, the results flipped my hypothesis upside down.

Finding: Stock Volatility Granger-causes Narrative Shifts at a 2-week lag. The news doesn't drive the market. Market fear forces corporate PR to change the news.

Why This Matters

🧠

Unsupervised Intelligence

Most sentiment analysis relies on rigid, manual tagging. This project deployed BERTopic Architecture to force the AI to autonomously cluster 6,267 unstructured articles without a single human label.

📈

The Predictive Edge

Standard dashboards only look backward. This project utilizes a Vector Autoregression (VAR) statistical engine to prove causality and predict future narrative shifts before they happen.

💼

Business Acumen

Hardcore math is useless if executives can't read it. This project translates complex volatility metrics directly into a Real-Time Risk Gauge and an actionable corporate PR playbook.

🏗️

End-to-End Engineering

This is not a disjointed Jupyter Notebook. This project represents a complete, automated pipeline from raw GDELT data ingestion and algorithmic processing, all the way to a custom-built, interactive presentation layer.

🗄️

Verifiable Ground Truth

This is not a synthetic dataset. The pipeline queried the massive GDELT Global Database to programmatically extract 6,267 authentic news articles, ensuring the algorithms were trained on objective, real-world journalistic reality.

🛡️

Fault-Tolerant Architecture

Handling heavy data extraction requires robust infrastructure. The ingestion engine utilizes asynchronous processing and fault-tolerant retry logic to achieve a massive 98.3% extraction success rate over an unbroken three-year timeline.
